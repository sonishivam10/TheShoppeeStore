
# TheShoppeeStore

## E-Commerce Site for buying lifestyle products online. 

AppLink - http://theshoppee.herokuapp.com/

## Feature:
- SignIn/SignUp (Authentication)
- View Product Catalogue
- Filter on the basis of Category
- Add to Cart
- Make Payment


## Tech Stack:
 - Backend - Python3, Django
 - FrontEnd - HTML, CSS
 - Database - Sqlite3
 - Payment Gateway - Stripe
 - Deployement - Heroku
